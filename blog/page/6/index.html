Why and How We Construct Real Number System

Sep 30th, 2014 | Comments

Preface

My journey of teaching myself mathematics begins with learning the number theory, understanding why matrix is invented. Thus up to now, most of the effort was spent on matrix algebra and matrix calculus.

Recently, I am trying to understand wavelet theory. The main motivation behind it is to find a set of basis to approximate functions, which seems to involve a lot of analysis. Therefore, I beginning to teach myself analysis like a math major student from scratch.

The first hard concept I met is the construction of real number system. I skipped this concept the first time I am learning analysis. In retrospect, I mainly focused on the concept of peano axiom to construct natural number system then because matrix algebra mainly deals with integer. However, the main objective of wavelet theory is to find a set of basis that can approximate functions, regardless of continuous and discrete. There is no escape this time.

The articles about peano axiom is on another PDF file, which I haven’t sorted it out on the blog. Some time in the future.

I guess this will be the beginning of a series of post about analysis. The first one here is about construction of real number system. Let us begin.

Some Words

The purpose of this post is to explain the intuition behind to construct the real number system, so it will not be mathematically sound and incremental. You should be familiar with the following concept:

  1. (ordered)field
  2. ordered set
  3. some knowledge about set and set operation
  4. Cauchy sequence
  5. Archimedean property

Motivation Sketch

The following are key points that motivate mathematicians to construct a new number theory that is called real number system.

  1. There are numbers that rational number cannot incorporate. In other word, there is gaps in rational number line.
  2. It may be surprising that for some $E \subset Q$, you cannot find the extreme point of $E$ in $Q$. This point is somehow a rephrase of the previous point – there are gaps and least upper bound may residue in the gap.

Therefore, to make the number line to be without gaps, in our daily word, continuous, the real number system is constructed.
