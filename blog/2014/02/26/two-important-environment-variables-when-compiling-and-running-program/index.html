Two Important Environment Variables When Compiling and Running Program

Feb 26, 2014

This post notes down two environment variables, LD_FLAGS and LD_LIBRARY_PATH which is essential to compile and run program under linux. I think there must be some Bible talking about this systematically, however, linux newbies just cannot find the them easily. So, it may be better to note it down.

LD_FLAGS

This variable is used by compiler to search for shared libraries when compiling program. See the post on compiling vim for an example.

LD_LIBRARY_PATH

This variable is used by dynamic shared lib loader to load the shared lib on the fly when you are running binary executable files. For example, to make shared in ./lib of current folder searchable when running SOME_PROGRAM, the command is like this:

env LD_LIBRARY_PATH="$LD_LIBRARY_PATH:./lib" ./SOME_PROGRAM
